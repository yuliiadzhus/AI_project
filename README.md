Назва та мета проєкту
TravelAI — це вебдодаток на базі штучного інтелекту, створений для швидкого планування бюджетних подорожей. Проєкт використовує модель Gemini 2.5 Flash для аналізу міст, надаючи користувачеві структуровану інформацію про безкоштовні локації та способи економії. Основна ідея полягає у створенні персоналізованого звіту, який дозволяє швидко зорієнтуватися в новому місці без зайвих фінансових витрат.

Опис цільової аудиторії
Продукт розроблений для мандрівників, які прагнуть самостійно планувати маршрути та мінімізувати витрати. Основну групу користувачів складають студенти, молодь та самостійні туристи, які зацікавлені у пошуку локальних продуктів і безкоштовного дозвілля.

Технологічний стек
Frontend: HTML5 та CSS для створення адаптивного інтерфейсу.
Backend: JavaScript та асинхронні запити Fetch API.
AI Core: Інтеграція з Google Gemini API для генерації аналітичних звітів.

Інструкція із запуску проєкту
Для запуску системи необхідно розмістити файли index.html, style.css та script.js в одній директорії. Після цього слід отримати персональний ключ у Google AI Studio та додати його у файл script.js.

Процес запуску включає наступні технічні кроки:
1. За наявності регіональних обмежень увімкніть VPN із локацією у США або Європі.
2. У налаштуваннях VPN-розширення активуйте доступ до URL-адрес файлів, щоб уникнути блокування запитів.
3. Відкрийте index.html у браузері, введіть назву міста та запустіть аналіз.

 Обробка технічних викликів (QA)
Під час розробки було вирішено проблему взаємодії локальних файлів з API в обхід обмежень CORS. Також реалізовано систему перевірки доступності сервісу при регіональних лімітах. Окремо додано обробку помилки 429 (Too Many Requests), яка інформує користувача про вичерпання лімітів безкоштовного рівня та вказує точний час очікування до наступного запиту.
